# Cuckoo Network

**Cuckoo Network = Cuckoo AI 去中心化推理 + Cuckoo Chain**

- **Cuckoo Chain** 是用于去中心化 AI 的 Arbitrum Orbit Rollup
- **$CAI**，Cuckoo Chain 的本地 Gas 代币
